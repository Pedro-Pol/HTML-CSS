Iframes

Os elementos iframe, inline-frame, são quadros nos quais são exibidos outros documentos, próprios ou de outro endereço da internet. Esses elementos recebem esse nome pois são impressos como elmentos inline-block. Anteriormente, foram empregados dois desses elementos para exibir vídeos do Youtube e do Vimeo.

O elemento iframe requer uma tag de abertura e outra de fechamento, no qual o atributo src, source, deve ser preenchido. Qualquer texto inserido entre as tags será exibido, caso o arquivo relacionado não possa ser exibido. 

<iframe src="" frameborder="0"></iframe>

O atributo scrolling serve para alterar a rolagem da página carregada no iframe. O valor yes, possibilita a rolagem, mesmo sem necessidade; o valor no impossibilita a rolagem, mesmo havendo tamanho suficiente para existir rolagem; e auto, que permitirá rolagem, apenas se houver tamanho para tanto.

• Navegação no iframe

É possível criar uma janela com o elemento iframe para acessar múltiplas outras páginas, empregando, para isso, links. Basta incluir o atributo name, no iframe, deixando o atributo src sem nenhum valor; e atribuir o mesmo valor concedido ao name, no atributo target dos links.

Po exemplo, considere clicar nos seguintes links, observando os efeitos causados no iframe abaixo:

  • Documento de exemplo I;
  • Documento de exemplo II;
  • Documento de exemplo III.

No exemplo acima, foi empregado o seguinte código:

<ul>
    <li>
        <a href="pag-extra/pag01.html" target="tela">
            Documento de exemplo I;
        </a>
    </li>
    <li>
        <a href="pag-extra/pag02.html" target="tela">
            Documento de exemplo II;
        </a>
    </li>
    <li>
        <a href="pag-extra/pag03.html" target="tela">
            Documento de exemplo III.
        </a>
    </li>
</ul>

<iframe src="" name="tela"></iframe>

Atributo srcdoc

Quando uma janela do iframe é carregada, ela pode exibir outro documento indicado no valor src, source; o que é mais convencional. Vale ressaltar, no entanto, que o atributo srcdoc pode ser empregado para escrever código HTML, como valor.

É necessário tomar o cuidado de não realizar quebras de linha e todas as aspas normais, compostas, tornam-se aspas simples.

Por exemplo, o exibido automaticamente no iframe acima inserido foi transcrito da seguinte maneira: 

<iframe srcdoc="<h1 style='text-align:center;
font-variant:small-caps;color:white;font-family:cambria;
margin:80px 0px 30px 0px'>Clique em um dos links acima</h1>
<p style='width:350px;text-align:justify;color:white;
margin:auto;letter-spacing:0.5px;font-size:1.2em;
font-family:cambria;'>Experimente clicar em um dos links
listados acima! Eles reproduzirão outros documentos .html no
interior da janela desse <strong>iframe</strong>.
</p>" name="tela"></iframe>
