Formulário

Formulários são todos os componente de HTML que interagem com o usuário, para que esse possa inserir dados; seja transcrevendo texto ou números, seja optando por alternativas disponíves. Os principais elementos são <form>, <input>, <select>, <datalist> e <textarea>. Além disso, a legenda, que denomina cada espeço do formulário é denominada pelo elemento <label>.

O elemento <form>

O formulário, <form>, abrange todos os demais itens de formulário, entre seu início e encerramento, envelopando-os. Seus atributos são: (i) action, cujo valor indica qual documento de linguagem de programação processará os dados inseridos; (ii) method, que determina o método de envio; sendo get e post seus valores. Toda etiqueta possuirá um atributo indicando a qual campo corresponde. Esse atributo é o for, e o seu valor deve ser igual ao id do elemento de formulário ao qual a etiqueta faz referência.

Além disso, é possível atribuir uma divisão interna aos formulários muito grandes, destacando a semelhança dessas informações. Para isso, emprega-se o elemento <fieldset>, que criará uma caixa contendo aquelas informações. É possível, ainda, atribuir uma legenda a esse campo de informações com o elemento <legend>.

O elemento <label>

Os campos de formulários, normalmente, possuem um texto imediatamente adjacente indicando com qual informação o campo deve ser preenchido. Em um primeiro momento, poder-se-ia pensar em adicionar um parágrafo ou outro elemento de texto inline; no entanto, para conferir uma denominação a um elemento de formulário emprega-se uma etiqueta, o elemento <label>.

O elemento <input>

O elemtno <input> possui vários tipos, sendo expressos por seu atributo, e correspondente valor, type. Estes são alguns dos mais relevantes:

  • text:
  • number:
  • password:
  • checkbox;
  • radio – estabelece alternativas com as quais o usuário poderá escolher apenas uma opção. Para tanto, é necessário criar vários elementos input com o mesmo valor transcrito no atributo name;
  • time;
  • date;
  • month;
  • color;
  • range;
  • file;
  • submit;
  • reset.

Atributos

Alguns atributos relevantes do elemento <input> são: required, que torna o campo de formulário obrigatório, não podendo enviar dados sem o seu preenchimento; minlength e maxlength que estabelecem o número mínimo e máximo, respectivamente, de caracteres que devem ser observados ao preencher um campo; size, que indica qual o tamanho da caixa de texto, conforme o número de caracteres cabíveis; placeholder, que exibe um texto padrão no campo, indicando sobre o que se trata, que logo desaparece assim que a primeira letra ou número é inserido; autocomplete, que indicará ao histórico do navegador qual o tipo de dados deverá autopreencher aquele campo em particular. Em campos de formulário de mútlipla escolha, o campo pode estar marcado, como padrão, introduzindo o atributo checked.

Em formulário numéricos, os atributos min e max estabelecem entre qual intervalo um campo deve ser preenchido; set determinará de quantos em quantos passos os números aumentam ou diminuem, conforme as setas auxiliares à direita.

O elemento <select>

O elemento <select> possibilita disponibilizar ao usuário um campo com múltiplas alternativas, empregando para isso o elemento <option>, escolhendo apenas um entre as diponíveis. É possível ainda, determinar, entre as alternativas, grupos de alternativas comuns com o elemento <optgroup>, nomeando cada grupo com a inserção do valor do atributo label.

O elemento <datalist>

O elemento <datalist> é empregado conjuntamente com um elemento <input>, para sugerir uma série de opções, sem excluir a possibilidade do usuário preencher o campo, transcrevendo-o como desejar. É um bom item de formulário para sugerir; ao invés de demandar a escolhar de uma lista exaustiva. O valor do atributo id da <datalist> deve ser o mesmo que o valor do atributo list, no <input>.

Diferentemente do elemento <option> dentro de <select>, o elemento <option> dentro do <datalist> não terá tag de fechamento, exibindo aquilo que for introduzido como valor do atributo value.

O elemento <textarea>

Como o nome sugere, trata-se de uma área de texto, com atributos cols e rows para definir o tamanho da janela de texto.

O elemento <output>

O elemento <output> é um componente de HTML desenvolvido para responder à linguagem de programação atuante no documento. É literalmente, uma saída para o que for programado no script do documento.
