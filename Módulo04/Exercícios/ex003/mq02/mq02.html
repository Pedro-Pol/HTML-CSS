Media Queries

As media queries – algo como requerimentos de mídia – são a combinação entre media types e media features; tipos e características de mída, respectivamente. Essa combinação orienta o documento ser exibido com diferentes estilos, orientado por muitos documentos de CSS, dependendo do tipo de mídia em que é exibida (impressão, telas, entre outros) e as características dessas mídas (considere o tamanho de telas de celular, telas de computador, telas de TV).

Em termos de código, as media queries são transcritas como valor no atributo media do link dos documentos de CSS. Os media types são transcritas em suas categorias; enquanto que as media features são transcritas entre parênteses. É possivel usar combinadores para declarar simultaneamente ambas as mídias: and, or, etc.

Veja como as media queries foram declaradas neste documento: 

<link rel="stylesheet" href="css/generico.css"
    media="all">
<link rel="stylesheet" href="css/landscape.css"
    media="screen and (orientation: landscape)">
<link rel="stylesheet" href="css/portrait.css"
    media="screen and (orientation: portrait)">
