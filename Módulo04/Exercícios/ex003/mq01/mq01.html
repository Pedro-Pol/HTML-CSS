O que são?

As media queries – algo como requerimentos de mídia – são diferentes apresentações do conteúdo do documento, que varia conforme o dispositivo em que será apresentado.

O código correspondente

Para criar media queries basta criar uma série de documentos de estilo que reponderão a determinadas circunstância. Além disso, é necessário introduzir o atributo media no link das CSS. Este documento, por exemplo, foi feito para possuir duas apresentações: (i) uma genérica de tela – valor screen – e (ii) outra para impressão – valor print –. confira o código: 

<link rel="stylesheet" href="01-01.css" media="screen">
<link rel="stylesheet" href="01-02.css" media="print">

Para testar, pressione Ctrl + P e veja as diferenças de estilo exibidas em ambos os documentos. Por exemplo, o menu meramente ornamental acima é desnecessário na maioria das impressões, pois o usuário do site está salvando um documento para consevar o conteúdo e não menus e outros itens típicos de um endereço da internet.
