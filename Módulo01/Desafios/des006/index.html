Tag em HTML5

Em HTML5, é possível inserir os elementos da listagem seguinte. Passe o mouse sobre os elementos listados é confira qual tag é necessária inserir para que o elemento seja inserido.

  • Parágrafo;
  • Títulos;
  • Linha Horizontal;
  • Destaque / Negrito;;
  • Ênfase / Itálico;
  • "Inserido" / Sublinhado;
  • Marca-texto;
  • Desconsiderar / Riscado;
  • Sobrescrito e Subscrito;
  • Pequeno e Grande;
  • Abreviações;
  • Imagens;
  • Áudios;
  • Vídeos;
