Elementos Básicos e Símbolos

Parágrafos e Quebra de Linha

Um dos elementos mais basilares do conteúdo de um documento HTML é o parágrafo. É possível transcrever um texto no interior das tags <p> e </p>. Isso atribuirá uma série de características de estilo, como margens, automaticamente. Já para inserir uma quebra de linha, basta empregar a empty tag <br>.

A maioria dos elementos de HTMl requer um par de tags, uma de abertura e outra de encerramento (como o elemento <p> e </p>). Agora, existem os elementos que dispensam o encerramento. Essa tags são denominadas empty tags, ou marcas vazias (como o elemento <br>).

Símbolos e Caracteres Especiais

Inicialmente, é nessário conhecer o Consórcio Unicode, que desenvolveu um catálogo de todos os caracteres; alfabéticos, numéricos, simbólicos ou diacríticos de todas as linguagens conhecidas. Cada caracter recebeu um nome numérico, um código. Esse código pode ser lido pelo programa executor do documento. Cada caracter possui um código de base decimal e de base hexadecimal. Os códigos unicode são distinguidos por serem escritos com o prefixo U+ seguido de algarismos que identificam o glifo. Para que seja lido por um arquivo HTML, o prefixo inicial é mudado para &#, para código decimal; e &#x para código hexadecimal. Ambos são encerrados por ponto e vírgula (;).

Além disso, o código hexadecimal de uma letra, diacrítico ou símbolo pode ser empregado em CSS. Neste caso, é necessário alterar o prefixo do código para \.

Além disso, a HTML mantém um pequeno léxico de entidades HTML que reduzem o código de determinados caracteres a uma abrevição alfabética. Por exemplo, o símbolo ⇒ é transcrito como &rArr;, do inglês right arrow: flecha para direita. Isso busca facilitar o emprego dos símbolos e caacteres mais correntes. Por exemplo, todos os colchetes angulares empregados no texto deste documento foram criados utilizando &lt; e &gt;. As entidades HTML também são encerradas por ponto e vírgula.

Veja a seguir alguns exemplos:

    Símbolo de Direitos Autorais — ©

  • &copy; — Entidade HTML;
  • &#169; — Código decimal;
  • &#x00A9; — Código hexadecimal;

    Símbolo de Marca Registrada — ™

  • &trade; — Entidade HTML;
  • &#8482; — Código decimal;
  • &#x2122; — Código hexadecimal;

    Símbolo de Registro de Patente Comercial — ®

  • &reg; — Entidade HTML;
  • &#174; — Código decimal;
  • &#x00ae; — Código hexadecimal;

A simples inserção desses códigos serão lidos pelo documento HTML como o símbolo a ele correspondente. Importante: nem todo símbolo ou caracter possui uma entidade HTML correspondente, mas todos possuirão código unicode, decimal e e hexadecimal.

Emojis

O Consórcio Unicode também abrange emojis. É importante firsar, no entanto, que dependendo do navegador e da atualização do sistema operacional, apenas uma ou outra versão de um emoji pode ser lida. Versões antigas do sistema operacional não permiter a atualização mais recente dos navegadores, o que reproduz emojis em um versão já ultrapassada.

Depois de consultar a listagem de emojis da Unicode e selecionar o desejado, copie o código unicode, com prefixo U+, e o substitua pelo prefixo &#x; pois todos os emojis são codificados em base hexadecimal.
