Listas em HTML

Para que seja criada uma lista em HTML, emprega-se as tags <ol> e <ul>. Respectivamente, são apreviações para ordered list e unordered list; no qual no primeiro caso haverá marcadores especificando a ordem daquele item. Em relação aos itens dessas listas, eles também são envelopados na tag <li>, isto é, listed item.

Exemplo:
"O conteúdo do livro abrangia os seguintes tópicos:

  1. Introdução
  2. Histórico
  3. Propedéutica
  4. Principiologia
  5. Tecnica e Normas"
"Escrevo a lista de supermercado desse mês:
  • Granola;
  • Iogurte;
  • Rúcula;
  • Gergelim."

Parâmetros e Atributos

Uma lista ainda pode receber os seguintes parâmetros e atributos: (a) type, indicando qual tipo dos marcadores na lista; que, em listas ordenadas, as opções são "1", para números (definição padrão); "A", para ordem alfabética maiúscula; "a", para ordem alfabética minúscula; "I" para algarismos romanos maiúsculos; e "i" para algarismos romanos minúsuculos; enquanto que, em listas não ordenadas, as opções são disc, uma bolinha preta (definição padrão); circle, um círculo sem preenchimento; e square, uma quadrado preenchido. Outro parâmetro é o (b) start que indica por qual número uma lista ordenada deve começar. Além disso, há o atributo (c) reversed que inverte as listas ordenadas, tornando-as decrescentes.

Exemplo:

"O conteúdo do livro abrangia os seguintes tópicos:

  1. Introdução
  2. Histórico
  3. Propedéutica
  4. Principiologia
  5. Tecnica e Normas"
<ol type="I">

"Escrevo a lista de supermercado desse mês:
  • Granola;
  • Iogurte;
  • Rúcula;
  • Gergelim."

<ul type="circle">

Lista Aninhadas

As listas podem ser inseridas umas dentro das outras, formando um conjunto de subitens listados, mais complexo e que atende às demandas mais específicas. Basta recordar-se de que a nova lista deve ser introduzida dentro e antes do fechamento da tag <li>.

Exemplo:
"O conteúdo do livro abrangia os seguintes tópicos:

  1. Introdução
  2. Histórico
    1. Concepção Antiga
    2. Concepção Renascentista
    3. Concepção Contemporânea
  3. Propedéutica
  4. Principiologia
    1. Princípios Processuais
    2. Princípios Materiais
  5. Tecnica e Normas
    1. Direito Internacional
    2. Ordenamento Jurídico
    3. Jurisprudência
      • Decisões do Tribunais
      • Jurisprudência do STJ
      • Jurisprudência do STF"

Anotações Adicionais

Atributo obsoleto denominado compact diminuia o espaço entre as linhas de uma listagem, normalmente, para satisfazer normas de formatação acadêmica, nas quais isso é requisitado. Como está obsoleto, prefere-se o comando das CSS style="line-height:100%", na qual a porcentagem indicada é o espaçamento entre linhas.

Exemplo:
"O conteúdo do livro abrangia os seguintes tópicos:

  1. Introdução
  2. Histórico
    1. Concepção Antiga
    2. Concepção Renascentista
    3. Concepção Contemporânea
  3. Propedéutica
  4. Principiologia
    1. Princípios Processuais
    2. Princípios Materiais
  5. Tecnica e Normas
    1. Direito Internacional
    2. Ordenamento Jurídico
    3. Jurisprudência
      • Decisões do Tribunais
      • Jurisprudência do STJ
      • Jurisprudência do STF"

Listas de Definição

À maneira de um dicionário, as listas de definição, contidas pela tag <dl>, definition list, possui termos individuais, contido pela tag <dt>, definition term, com descrições correspondentes envelopados pela tag <dd>, definition description. Embora a formatação não seja alterada substancialmente, ela associa os termos às suas descrições, auxiliando os mecanismos de busca.

Exemplo (CSS locais modificadas):
