FONTES EM CSS

DECLARAÇÃO FONT-FAMILY

É possível determinar um grupo de fontes, simultaneamente, para compor a escolha estilística do documento. Essa declaração é feita da seguinte forma:

body {
font-family: Arial, Helvetica, serif;
}

Nesta declaração, informa-se a máquina que a fonte Arial deve ser selecionada. Caso, não a encontre nos bancos de dados do dispositivo leitor, tentará a próxima que, neste caso, é a fonte Helvética. Caso também não seja encontrada a segunda opção, então, é indicada um categoria de fonte, que é a das serifadas, neste caso.

Estes conjuntos possuem fontes semelhantes e têm a função de assegurar que o aparelho leitor consiga reproduzir um resultado estético idêntico ou parecido ao projeto finalizado. As combinações já testadas e conhecidas recebem a expressão de web safe font combinations ou fallback fonts. O W3Schools mantém uma lista compreensiva destas combinações.[acesse aqui]

DECLARAÇÃO FONT-SIZE

É possível alterar o tamanho das fontes, empregando uma declaração em especíico, a seguinte:

body {
font-family: Arial, Helvetica, serif;
font-size: 1.2em;
}

Nesta declaração, é possível inserir medidas absolutas e medidas relativas. As primeiras são compostas pelas seguintes unidades: cm (centímetros), mm (milímetros), px (pixels), pt (pontos) e pc (paicas); em que os dois últimos são unidades criadas para impressão e não telas. As medidas relativas são as seguintes: em (emphemeral unit, que representa a largura da letra maiúscula M, na fonte referenciada), ex (ephemeral x-height, que representa a altura-x, na fonte referenciada), rem (root ephemeral unit), Vw (view width), Vh (view height) e % (porcentagem).

A WC3 recomenda o emprego preferencial dos pixels, como unidade absoluta, e, principalmente, da ephemeral unit, como unidade relativa; considerando o desempenho e regularidade apresentado em diversos dispositivos.

Elaborando mais: a ephemeral unit é uma relação de proporção com o tamanho da letra maíuscula M da fonte selecionada. A medida 1em significa 100% dessa medida, enquanto que 1.2em significa 120% dessa medida, e assim em diante.

DECLARAÇÃO FONT-WEIGHT

É possível, em algumas fontes, estabelecer o peso da fonte, isto é, o quão largas são seus glifos, assemelhando-se a variações maiores ou menores de negrito. A declaração que torna isso possível, em fontes com essa variação é a seguinte:

body {
font-family: "work Sans", sans-serif;
font-size: 1.2em;
font-weight: lighter;
}

É possível alterar o peso nominal ou numericamente. Nominalmente, existem os comandos: lighter, normal, bold e bolder. Numericamente, o peso é determinado com valores entre 100 e 900.

FONT-STYLE & TEXT-DECORATION

É possível indicar itálico, sublinhado, riscado, sobrelinhado, entre outros, com as seguintes declarações:

body {
font-family: "Work Sans", sans-serif;
font-size: 1.2em;
font-weight: lighter;
font-style: italic;
text-decoration: underline;
}

SHORTHAND DA DECLARAÇÃO FONT

É possível condensar as quatro declarações de fonte, em uma única declaração, desde que observada a ordem font-style, font-weight, font-size e font-family. Se fosse reduzida a essa declaração única, o exemplo anterior estaria resumido na expressão:

body {
font: italic lighter 1.2em "Work Sans", sans-serif;
}
