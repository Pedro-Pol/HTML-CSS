IMPORTAR FONTES

É natural que as fontes pré-instaladas no dispositivo não satisfassam todas as demandas do desenvolvimento de portais. Portanto é necessário baixar novas fontes.

REGRA @import

É possível fazer referência a um link externo, no qual o documento buscará a fonte indicada. Para tanto, é necessário transcrever um regra de CSS, que são postas no topo das folhas e antecedidas por arroba. Neste caso, a regra é @import url(''). Neste exemplo, foi encontrada a fonte no Google Fonts

Em uma circunstância hipotética, quer-se empregar a fonte Barrio para um detemrinado título. Devido a essa fonte não estar presente no repositório do dispositivo, anexou-se assim uma ligação externa:

@import url('https://fonts.googleapis. com/css2?family=Barrio&display=swap');

h1#special {
font-size: 3em;
font-family: 'Barrio', cursive;
} [...]

REGRA @font-face

A segunda possibilidade é fazer o download de um arquivo que contenha uma fonte. Os formatos desses arquivos podem ser: .otf (open type font); .ttf (true type font); .svg (scalable vector graphics). É necessário acrescer uma regra de CSS: a @font-face. Isso permitira indicar uma URL que contenha o arquivo de fonte desejado. Além disso, permite nomear a fonte dentro do documento, que correspondera ao valor inserido na propriedade font-family.

Em uma circunstância hipotética, é desejável incluir uma fonte, semelhante a anterior, no entanto, que cubra texto normal, isto é, que inclua minúsculas. A fonte Barriecito satisfaz essa demanda. O código inserido seria o seguinte:

@font-face {
font-family: Barriecito;
src: url('fonts/Barriecito-Regular.ttf') format('truetype');
font-size: normal;
font-weight: normal;
font-style: normal;
}

p#special {
font-family: Barriecito, sans-serif;
font-size: 1.3em;
}

O resultado seria algo como:

Regras, declarações, propriedades e valores em CSS

As propriedades e valores adicionais declados dentro da regra, indicam como a fonte deve ser lida, especialmente quando o arquivo de fonte contém variações de peso (como mencionada, a variação de negrito).

Além disso, algumas fontes possuem símbolos que podem ser empregados com outras finalidade estéticas. Para exemplificar, é indicada a fonte Agathodaimon:

pae fsn dor

Ou ainda, a fonte Numismatic:

b c d
