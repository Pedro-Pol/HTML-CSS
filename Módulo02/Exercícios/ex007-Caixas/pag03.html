Cabecalho

Para criar o cabeçalho, emprega-se a tag <header>; no qual, normalmente, serão transcritos o título do portal e o índice de navegação. Opcionalmente, lemas e outras ordens de subtítulos também podem ser adicionados.

link link link link

Para cria o índice de navegação, emprega-se a tag <nav>, no qual serão transcritos os menus temáticos de navegação e seus links correspondentes.

Conteudo

A principal parte de um portal é destacada por conter o conteúdo do documento. Para criar essa porção especial, emprega-se a tag <main>. Como mencionado, nesta porção, será dedicada ao conteúdo principal a que se dedica o portal.

Seccao I

A distinção entre secções e artigos é arbitrária. É possível desenvolver artigos dentro de secções, assim como secções dentro de artigos. O emprego de ambos depende da necessidade. Para criar uma secção, emprega-se a tag <section>.

artigo

Agora, para criar um artigo, basta empregar a tag <article>. Conforme mencionado anteriormente, a distinção entre os dois elementos é arbitrário, com pouco implicação semântica.

artigo

A ideia mais importante de entender é que ambas os elementos podem ser criados uns dentro dos outros, em um aninhamento de conteúdo.

Artigo
