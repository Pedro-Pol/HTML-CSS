CUSTOMIZACAO DE CAIXAS

Sombreamento de Caixas

É possível adicionar um efeito gráfico que simula uma sombra, empregando, para isso, a propriedade de CSS box-shadow. Os valores inseridos nesta propriedade são cinco, na seguinte ordem: (i) deslocamento horizontal; (2) deslocamento vertical; (3) embassamento; (4) espalhamento; e (5) a cor da sombra. Além disso, a propriedade, por padrão, projeta uma sombra externa, outset; todavia, é possível, declarando como primeiro valor o contrário, isto é: inset, uma sombra interna.

Neste documento foi inserida uma sombra no elemento <main>. O código empregado foi o seguinte: 

    main {
        width:      450px;
        box-shadow: 5px 5px 5px hsla(0, 3%, 14%, 0.275);
    }

É interessante ressaltar o emprego de uma cor com o vetor alpha, isto é, onde é aplicada a transparência. Isso torna mais suave o impacto exagerado de uma cor sólida.

Vertices Arredondados

É possivel aredondar os vértices, os cantos, das caixas. Para tanto emprega-se a propriedade border-radius. Essa propriedade, em si, é um shorthand para quatro propriedades, nesta ordem: (i) border-top-left-radius; (ii) border-top-right-radius; (iii) border-bottom-right-radius; e (iv) border-bottom-left-radius. E, assim como os shothands de margin e padding, os valores podem ser inseridos de três modos: (a) abrangendo todos os quatro valores, no orderm supramencionada; (b) dois valores, em que o primeiro valor determina os vértices superior esquerdo e inferior direito; enquanto que o segundo valor determina os vértices superior direito e inferio esquerdo; e (c) um único valor que será aplicado a cada vértice.

Esse documento possui o elemento <main> com a propriedade de arredondamento de vértices. Esse é o código empregado para fazê-lo: 

    main {
        border-radius:  15px;
    }
