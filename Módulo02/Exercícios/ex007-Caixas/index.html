Caixas

Todos os elementos de HTML estão encerrados em quadradados, denominados caixas. Entender as características dessas caixas é essencial para manipulá-las em CSS.

CARACTERISTICAS DE CAIXAS

O conteúdo de um elemento HTML está encerrado entre um quadrado com duas dimenções, a altura  height  e a largura  width . Imediatamente posterior a esses limites está a borda  border . Ao espaçamento entre a borda e o conteúdo, o espaço interior, é concedido o nome de acolchoamento  padding ; e, ao espaçamento entre a borda e os outros elementos adjcentes, o espaço exterior, é concedido o nome de margem  margin . Também há o outline, o contorno, que é uma área no interior da margem, em divisa com a borda.

SHORTHANDS

A margin e o padding possuem propriedades parecidas; assim como border e o outline. Observe: 

    div#box {
        margin-top:     25px;
        margin-right:   20px;
        margin-bottom:  15px;
        margin-left:    10px;

        padding-top:    10px;
        padding-right:  15px;
        padding-bottom: 20px;
        padding-left:   25px;

        border-width:   10px
        border-style:   dotted;
        border-color:   red;

        outline-width:  10px
        outline-style:  solid;
        outline-color:  blue;
    }

Existem shorthands  isto é, reduções de código  semelhantes para propriedades semelhantes. Observe o exemplo que conserva os valores do exemplo anterior:

    div#box {
        margin:     25px 20px 15px 10px;
        padding:    10px 15px 20px 25px;

        border:     10px dotted red;
        outline:    10px solid blue;
    }

A margin e o padding ainda possuem um shorthand ainda mais sumário: (a) se houver apenas dois valores, para os intervalos verticais e outro para os horizontais, nesta ordem; e (b) se os valores, tanto dos intervalos verticais ou horizontais, forem os mesmos. Observe: 

    div#box {
        margin:     20px 10px;
        padding:    15px 25px;

        margin:     15px;
        padding:    20px;
    }

Classificacao de Caixas

Ainda, as caixas podem ser classifcadas como inline-level ou box-level. No primeiro caso, a inserção do elemento classificado como inline-level não cria nenhuma quebra de linha. Do modo contrário, os classificados como box-level criam um espaço entre elementos, separando-os por uma quebra de linha.

ALTERACOES

É possível, ainda, alterar a característica de um elemento como box-level ou como inline-level. Para isso, basta empregar a propriedade display, acrecendo os valores: (i) block, para torná-lo um elemento box-level; (ii) inline, para torná-lo um elemento inline-level; ou ainda (iii) inline-block, para configurá-lo como um elemento inline-level com propriedades modificáveis de elemento box-level, como height e width.
