Seletores Personalizados

É possível atribuir qualidades específicas a determinados elementos do documento, para que, quando alteradas as propriedades de estilo, apenas as tags atribuídas respondam a tais mudanaças. Isso é feito operando seletores personalizados, quais sejam: identificadores, classes, pseudo-identificadores e pseudo-classes.

Identificadores

É possível identificar um elemento do documento HTML atribuindo-lhe o atributo global id, complementado por qualquer valor desejável. Nas CSS, esse identificador é demarcado por uma cerquilha  o símbolo # , seguido do denominador, o valor escolhido.

Por exemplo, o título principal de um documento  texto esse envelopado por uma tag <h1>  é de especial importância estética para a apresentação de qualquer portal eletrônico. Portanto, é desejável alterar-lhe o estilo, tornando-o mais chamativo. No entanto, também não é desejável alterar todos os elementos envelopados nas tags <h1>. Para tanto, atribuiremos o atributo global id para o título dessa página, com o valor principal. 

<h1 id="principal">Seletores
Personalizados</h1>

Em um segundo momento, realiza-se as mudanças nas CSS, indicando a tag, a cerquilha e o valor com o qual foi nomeado. Neste caso, a alteração para um fonte mais chamativa, procedendo às seguintes alterações no código: 

#principal
    {
        font-family:   'Yeseva One',
        cursive;
        text-indent:   0px;
        margin-top:    30px;
        margin-bottom: 30px;
        text-align:    center;
    }

Classes

É possível classificar um elemento HTML introduzindo um atributo global denominado class, complementando-o com um valor qualquer desejado. Semelhante ao identificador, sua função é conferir propriedades particulares de estilo aos elementos classificados com esse atributo. Diferente do identificador, a classe pode ser empregada várias tags, iguais ou diferentes.

Por exemplo, em algumas porções do texto deste documento, que foram configuradas para transcrever o código-fonte, foi empregada a classes. Veja: 

<p class="codigo">
    <h1 id="principal">Seletores
    Personalizados</h1>
</p>

Novamente, em um segundo momento, são configuradas as propriedade especiais desses trechos, isto é, desse parágrafos classificados pelo valor "codigo". 

    .codigo
    {
        text-indent:    0px;
        font-family:    monospace;
        font-size:      1.1em;
        margin-left:    30px;
        margin-top:     20px;
        margin-bottom:  20px;
    }

Importante salentar as diferenção entreos atributos id e class. Um identificador deve ser empregado para alterar apenas um único elemento em um documento, conforme conselho da W3C. Já os classificadores podem ser empregados multiplas vezes em múltiplos elementos de um mesmo elemento. Em dúvida, utiliza-se o segundo.
