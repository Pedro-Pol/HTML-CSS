Pseudo-Classes

As pseudo-classes governam o estado dos elementos de HTML. Por exemplo, os links possuem vários estados estéticos: se o link nunca tive sido acessado, mostra-se em cores azuis; se já acessado, em roxo; se o link possuir um IRL inexistente, vermelho. Essas modificações são feitas através pseudo-classes, que são demarcados, no documento CSS, por sinal de dois pontos (:) e seguido pelo termo que indica uma função específica.

Os links entre páginas, encontrados na porção inferior deste documento, foram modificados para sempre apresentarem-se no tom de marrom saturado, apresentando um tom avermelhado quando clicado. O código empregado para realizar tais alterações no padrão de apresentação dos links foi o seguinte: 

  a:link {
      color:            #7B6B43;
      text-decoration:  none;
  }
  a:visited {
      color:            #7B6B43;
      text-decoration:  none;
  }
  a:hover {
      color:            #7B6B43;
      text-decoration:  none;
  }
  a:active {
      color:            #7b4e43;
  }
    text-decoration:  none;

Ainda, é possível esconder uma passagem de texto no interior de outro, empregando sinais de maior que (>) ou de menor que (<), para indicar que um elemento está contigo  é um elemento child  dentro de um elemento que o contém  elemento parent.

Por exemplo, para ver o código fonte de como foi transcrita essa técnica, passe o mouse no seletor abaixo: 

        pre > span {
            display:   none;
        }
        
        pre:hover > span {
            display:   block;
        }

Pseudo-Elementos

Pseudo-elementos são seletores especiais que condicionam o conteúdo das tags. São demercados com um duplo dois pontos (::), e são seguidos do termo que especifica a função desejada.

Por exemplo, imaginemos que o portal encomendado tenha que realçar todo conteúdo novo postado recentemente com, digamos, um estrela. Uma seletor de pseudo-elemento do tipo ::after poderia ser criado para solucionar o problema.

Por exemplo:

  • Episódio 004
  • Episódio 005
  • Episódio 006
  • Episódio 007
  • Episódio 008
  • Episódio 009

Para o exemplo, foi empregado o seguinte código, respectivamente, em HTML e CSS: 

<ul>
    <li>Episódio 004</li>
    <li>Episódio 005</li>
    <li>Episódio 006</li>
    <li class="novo">Episódio 007</li>
    <li class="novo">Episódio 008</li>
    <li class="novo">Episódio 009</li>
</ul>

.novo::after {
    content: ' \2605';
}
