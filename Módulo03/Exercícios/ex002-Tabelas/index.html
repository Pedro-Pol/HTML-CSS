Tabelas em HTML

Semelhante ao elemento <dl>, description list, que contém os elementos <dt> e <dd>, description term e definition description; as tabelas são definidas pelo macro-elemento <table>, dentro do qual são definidas as linhas de tabelas nas tags <tb>, do inglês table row, que, por sua vez recebem os elementos <td>, table data, na qual está contida a informação da tabela.

Observe a tabela se seguir:

A1 B1 C1 D1
A2 B2 C2 D2
A3 B3 C3 D3
A4 B4 C4 D4

Para criar a tabela acima foram empregados os seguintes códigos, em HTML e CSS, respectivamente: 

    <table>
        <tr>
            <td>A1</td>
            <td>B1</td>
            <td>C1</td>
            <td>D1</td>
        </tr>
        <tr>
            <td>A2</td>
            <td>B2</td>
            <td>C2</td>
            <td>D2</td>
        </tr>
        <tr>
            <td>A3</td>
            <td>B3</td>
            <td>C3</td>
            <td>D3</td>
        </tr>
        <tr>
            <td>A4</td>
            <td>B4</td>
            <td>C4</td>
            <td>D4</td>
        </tr>
    </table>



    table {
        border-collapse: collapse;
        width:           250px;
        margin:          10px auto;
    }
    td {
        border:          1px solid black;
        padding:         8px;
    }
