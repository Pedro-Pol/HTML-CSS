Tabelas

No entanto, o modelo empregado anteriormente atende apenas tabelas simples. Tabelas complexas possuem elementos semânticos filhos especializados. No interior de <table>, podem ser inseridas o cabeçalho da tabela, o corpo da tabela e o rodapé da tabela; respectivamente, representados pelos elementos <thead>, <tbody> e <tfoot>. Dentro de cada componente, então, é inserida uma table row, isto é, uma linha, com o elemento <tb>. Dentro da linha, são inseridos dados com o elemento <td>, table data; e categorias com o elemento <th>, table head. Além disso, o título da tabela é inserido dentro do elemento <caption>.

Atributo scope

Há um atributo que define qual o escopo dos elementos de categoria, os <th>, indicando aos mecanismos de busca quais os dados que estão relacionados àquela categoria. Basta incluiro o atributo scope, definindo como valor col, de column, quando os dados são dispostos de maneira sotoposta, ou row, quando os dados relacionados estão dispostos de maneira justaposta.

Por exemplo, as categorias estado, porcentagem e população possuem um escopo colunar; enquanto total possui um escopo lateral.

Efeito zebrado

Para criar o efeito zebrado, que auxilia na leitura de tabelas, em uma tabela, emprega-se os seletores e operadores habituais, no entanto, com a adição de uma pseudo-classe: a :nth-child(). O conteúdo que preenche os parênteses indicará em que intervalo repetitivo as propriedades serão aplicadas. Por exemplo, :nth-child(2n) faz com que as propriedades sejam aplicas uma vez a cada duas repetições de determinado item. Ainda aceita os valores even e odd, isto é, par ou ímpar.

O efeito zebrado exibido na tabela acima foi gerado empregando o seguinte código: 

tbody > tr:nth-child(odd) {
    background-color: rgba(0, 0, 0, 0.1);
}

tbody > tr:nth-child(even) {
    background-color: white;
}

Efeito de cabeçalho flutuante

Para criar um efeito em que o cabeçalho flutuante, isto é, de um cabeçalho não fixo à extremidade da tabela e que segue o usuário conforme ele rola verticalmente a página, basta realizar as seguintes alterações:

  • Altere a posição da tabela para relativa;
  • Altere a posição do cabeçalho para "grudento", sticky;
  • Insira a propriedade top, com o valor adequado;
  • Reinsira outras propriedades já declaradas que devem mover-se junto com o conteúdo do cabeçalho, como, por exemplo, a propriedade background-color.

Esse efeito de cabeçalho flutuante foi inserido na tebala acima, conforme as seguintes declarações: 

table {
    border-collapse:  collapse;
    width:            400px;
    margin:           20px auto;
    position:         relative;
}

thead {
    position:         sticky;
    top:              -1px;
    background-color: aaaaaa;
}
