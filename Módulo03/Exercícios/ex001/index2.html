IMAGENS DE FUNDO

A propriedade background-image permite a inserção de um valor como uma URL, selecionando uma imagem interna ou externa.

A propriedade background-repeat permite alterar a repetição automática. Como mencionado, o valor padrão é repeat, sendo possível alterar para repeat-x, para repetição lateral; e repeat-y para repetição vertical.

A propriedade background-size possibilita inserir como valor as dimenções de comprimento e altura, respectivamente. Se for alterado apenas um número como valor, a imagem será redimencionada proporcionalmente.

Por exemplo, para o fundo desse documento, foram empregadas as seguintes configurações: 

    
        body {
            background-image:  url
                ('imagens/pattern002.png');
            background-size:   80px;
            background-repeat: repeat;
        }
