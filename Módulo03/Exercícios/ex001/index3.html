IMAGENS DE FUNDO

É possível empregar a propriedade background-position para determinar qual o ponto de ancoragem de uma imagem de fundo. Os valores aceitos são nove: right bottom; right center; right top; center bottom; center center – ou, simplesmente, center –; center top; left bottom; left center; e o padrão left top. Este documento, por exemplo, emprega o seguinte código: 

body {
    background-image: url
        ('imagens/wallpaper005.jpg');
    background-position: top center;
    background-repeat: no-repeat;
}

Novamente, é possível aplicar a propriedade background-image tanto no body quanto em qualquer outro elemento do documento. Uma maneira de demonstrar a aplicabilidade do conceito é refazer o desafio de mapa-mundi, com apenas uma imagem, alternando o ponto de ancoragem da propriedade mencionada.



Este é o código necessário para reproduzir o resultado acima, considerando que a imagem original possui 450 por 280 pixels: 


aside {
    display: block;
    margin: 20px auto;
    width: fit-content;
}

div {
    width: 145px;
    height: 89px;
    margin: 0px 0px;
    border: 3px solid 
        rgba(0, 0, 0, 0.548);
    display: inline-block;
    margin-bottom: -4px;
    margin-left: -5px;
}

div#tl {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: left top;
    border-top-left-radius: 5px;
}
div#tc {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: center top;
    margin-left: -6px;
}
div#tr {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: right top;
    border-top-right-radius: 5px;
}
div#cl {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: center left;
}
div#cc {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: center;
    margin-left: -6px;
}
div#cr {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: right center;
}
div#bl {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: bottom left;
    border-bottom-left-radius: 5px;
}
div#bc {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: bottom center;
    margin-left: -6px;
}
div#br {
    background-image: url
        ('imagens/mapa-mundi.png');
    background-position: bottom right;
    border-bottom-right-radius: 5px;
}
